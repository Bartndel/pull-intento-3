DESCRIPTION

--list: List all variables set in config file, along with their values.

--list --show-origin: ubicacion de las configuraciones guardadas.

--global: For writing options: write to global ~/.gitconfig file rather than the repository .git/config, write to $XDG_CONFIG_HOME/git/config file if this file exists and the ~/.gitconfig file doesnâ€™t.

For reading options: read only from global ~/.gitconfig and from $XDG_CONFIG_HOME/git/config rather than from all available files.

See also FILES.
